# 教育委員会向け PTA運営適正化ポータル

`index.html` をブラウザで開くと閲覧できます。外部CDNを使わない静的サイトとして作成しているため、ZIP展開後もそのまま動きます。

## 目的

PTA適正化推進委員会の公開資料をもとに、教育委員会の担当者が「何を通知し、何を点検し、どこから分離するか」を判断できる実務画面へ再構成しました。

## 主な内容

- 教育委員会向けの冒頭読み物本文
- PTA運営適正化の5原則
- 入学式・入学説明会・学校連絡ツール等の禁止例
- オプトイン／オプトアウトと「入会の記録」の説明
- 文部科学省の学校と教師の業務の3分類に基づく学校徴収金整理
- 非加入世帯の特定、協力金化、委託契約、職専免の論点整理
- 現状リスク一覧
- 適正化フロー
- 学校・PTA・保護者の境界線
- 実施ロードマップ
- 点検チェックリスト
- 学校長宛て通知文案
- 根拠資料リンク
